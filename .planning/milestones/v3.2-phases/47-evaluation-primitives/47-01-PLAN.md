       - Module docstring: "Pure-function evaluator for the detection stage."
       - `from __future__ import annotations`
       - Import `dataclasses`, `numpy`, `Detection` from core types
       - `@dataclass(frozen=True) class DetectionMetrics`:
         - `total_detections: int` — sum of all detections across all frames and cameras
         - `mean_confidence: float` — mean detection confidence across all detections
         - `std_confidence: float` — std of detection confidence
         - `mean_jitter: float` — mean absolute frame-to-frame delta in detection count per camera, averaged over cameras (per CONTEXT.md: 9→9→9→0→0→0 has low jitter ~1.5; 9→3→8→2→9→4 has high jitter ~5)
         - `per_camera_counts: dict[str, int]` — total detections per camera
         - `def to_dict(self) -> dict[str, object]`: explicit float() casts on all numeric fields, dict() on per_camera_counts
       - Helper `_compute_jitter(counts_per_camera: dict[str, list[int]]) -> float`:
         - For each camera, compute np.mean(np.abs(np.diff(counts)))
         - Return mean across all cameras; 0.0 if no cameras or all cameras have <2 frames
       - `def evaluate_detection(frames: list[dict[str, list[Detection]]]) -> DetectionMetrics`:
         - Input: list of per-frame dicts, each mapping camera_id to list of Detection objects. List order = frame order.
         - Accumulate total detections, all confidences, per-camera counts, per-camera per-frame detection counts
         - Call _compute_jitter for jitter metric
         - Return DetectionMetrics

    1. Create `src/aquapose/evaluation/stages/tracking.py`:
       - Module docstring: "Pure-function evaluator for the tracking stage."
       - `from __future__ import annotations`
       - Import `dataclasses`, `numpy`, `Tracklet2D` from core tracking types
       - `@dataclass(frozen=True) class TrackingMetrics`:
         - `track_count: int` — number of tracklets
         - `length_median: float` — median track length (number of frames)
         - `length_mean: float` — mean track length
         - `length_min: int` — minimum track length
         - `length_max: int` — maximum track length
         - `coast_frequency: float` — overall fraction of frames that are "coasted" across all tracklets
         - `detection_coverage: float` — fraction of (tracklet, frame) pairs that are "detected" (not coasted)
         - `def to_dict(self) -> dict[str, object]`: explicit float/int casts
       - `def evaluate_tracking(tracklets: list[Tracklet2D]) -> TrackingMetrics`:
         - Input: list of Tracklet2D objects from all cameras
         - Compute track_count = len(tracklets)
         - Compute lengths = [len(t.frames) for t in tracklets]; guard against empty
         - Compute coast_frequency = total coasted frames / total frames across all tracklets; guard div-by-zero
         - detection_coverage = 1.0 - coast_frequency (since every frame is either detected or coasted)
         - Return TrackingMetrics
