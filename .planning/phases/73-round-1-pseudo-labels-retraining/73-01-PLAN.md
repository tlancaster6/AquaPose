**Step 2: Create `src/aquapose/training/select_diverse_subset.py`.**

This standalone script reads pseudo-label directories and confidence.json sidecars, applies multi-axis diversity sampling, and copies selected files to a subset output directory.

**OBB selection (~50 images):**
1. Read `confidence.json` from `{pseudo_dir}/obb/`
2. Parse filenames: `stem[:6]` = frame_idx (int), `stem[7:]` = cam_id
3. Group entries by cam_id (expect 12 cameras)
4. Within each camera, divide frame indices into 4 equal temporal bins
5. From each bin, select 1 frame — tiebreaker: prefer frames with underrepresented `tracked_fish_count` values (compute global fish-count histogram, pick frame whose fish count has lowest global frequency)
6. Result: 4 per camera x 12 cameras = 48, plus pick 2 more from bins with highest temporal range to reach ~50
7. Copy selected images + labels to `{output_dir}/obb/images/` and `{output_dir}/obb/labels/`
8. Also copy a filtered `confidence.json` containing only selected entries

**Pose selection (~320 crops, consensus only):**
1. Read `confidence.json` from `{pseudo_dir}/pose/consensus/`
2. For each entry, extract per-fish `curvature_3d` from `labels` array. Each crop file is one fish, so match crop filename `{frame}_{cam}_{fish_idx}` to entries
3. Compute curvature quartile bins across all available crops
4. Within each quartile bin, sample proportionally across cameras (roughly equal camera representation within each bin)
5. Target 320 total. If fewer than 320 available, take all
6. For the secondary val set: among selected crops, identify the last ~20% by frame_idx (highest frame indices) and place them in `{output_dir}/pose/val/` subdirectory. Place the remaining 80% in `{output_dir}/pose/train/` — the `val/` subdirectory will cause auto-tagging with "val" during store import
7. Copy selected images + labels and filtered confidence.json
