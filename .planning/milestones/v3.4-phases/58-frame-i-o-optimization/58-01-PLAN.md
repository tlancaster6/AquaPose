**Part A — ChunkFrameSource prefetch (frame_source.py):**

Add imports: `import queue`, `import threading`. Add module-level `_SENTINEL = object()`.

Add instance attributes to `__init__`:
- `self._prefetch_queue: queue.Queue | None = None`
- `self._prefetch_thread: threading.Thread | None = None`
- `self._stop_event: threading.Event | None = None`

Replace `__iter__` with the prefetch pattern from RESEARCH.md:
1. Guard against concurrent iteration: if `_prefetch_thread` is not None and alive, raise `RuntimeError`.
2. Call `_ensure_captures_positioned()` (new method) — checks `CAP_PROP_POS_FRAMES` on each capture, seeks only if needed.
3. Create `queue.Queue(maxsize=2)`, `threading.Event()`, start daemon `threading.Thread(target=self._prefetch_worker)`.
4. In a try/finally: dequeue items, yield `(local_idx, frames)`, handle `_SENTINEL` (break) and `Exception` (raise).
5. In finally: set stop_event, drain queue, join thread with timeout=5.0.

Add `_prefetch_worker` method:
- Loops `local_idx` in `range(self.end_frame - self.start_frame)`.
- Checks `self._stop_event.is_set()` each iteration.
- For each camera: calls `cap.read()`. On `ret=False`, log warning and skip that camera (do NOT raise). On unexpected exception, put it in queue and return.
- Enqueues `(local_idx, frames)` using `put(item, timeout=0.5)` in a loop that checks `stop_event`.
- After loop, enqueues `_SENTINEL`.
- Wraps entire body in try/except: on unhandled exception, put the exception object in queue.

Add `_ensure_captures_positioned` method:
- For each camera, check `int(cap.get(cv2.CAP_PROP_POS_FRAMES))`. If != `self.start_frame`, seek and verify.
- Raise RuntimeError if seek fails.

Update `__exit__`:
- Set stop_event if exists.
- Drain queue if exists (get_nowait in a loop, ignore queue.Empty).
- Join thread if exists and alive, with timeout=5.0.
- Log warning if thread still alive after join.

Keep `read_frame` unchanged (seek-based random access). Add a docstring note that `read_frame` must NOT be called during active iteration.

Keep `__enter__` unchanged (no-op, returns self).
