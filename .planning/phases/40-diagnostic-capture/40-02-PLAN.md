Key scheme in midline_fixtures.npz:
- "meta/version" → string "1.0"
- "meta/camera_ids" → 1D object array of camera ID strings
- "meta/frame_indices" → 1D int64 array of frame indices
- "meta/frame_count" → scalar int
- "meta/timestamp" → string ISO timestamp
- "midline/{frame_idx}/{fish_id}/{camera_id}/points" → (N, 2) float32
- "midline/{frame_idx}/{fish_id}/{camera_id}/half_widths" → (N,) float32
- "midline/{frame_idx}/{fish_id}/{camera_id}/point_confidence" → (N,) float32

1. **Add `load_midline_fixture(path)` to `src/aquapose/io/midline_fixture.py`**:
   - Signature: `def load_midline_fixture(path: Path | str) -> MidlineFixture`
   - Open NPZ with `np.load(str(path), allow_pickle=True)`
   - **Validate metadata:**
     - Check `"meta/version"` exists; raise `ValueError("Missing meta/version in fixture file {path}")` if absent
     - Check version == NPZ_VERSION; raise `ValueError("Unsupported fixture version '{v}' (expected '{NPZ_VERSION}') in {path}")` if mismatch
     - Extract `frame_indices` from `"meta/frame_indices"`
     - Extract `camera_ids` from `"meta/camera_ids"`
   - **Parse midline keys:**
     - Iterate NPZ keys matching prefix `"midline/"`
     - Parse each key: split on `/` to get `["midline", frame_idx_str, fish_id_str, camera_id, field_name]`
     - Group by `(frame_idx, fish_id, camera_id)` to collect all 4 arrays (points, half_widths, point_confidence, is_head_to_tail)
     - For each group, construct `Midline2D(points=..., half_widths=..., fish_id=int(fish_id_str), camera_id=camera_id, frame_index=int(frame_idx_str), is_head_to_tail=bool(arr), point_confidence=...)`
     - Assemble into `MidlineSet` dicts per frame_idx
   - **Build MidlineFixture:**
     - Order frames by frame_indices
     - Build `metadata` dict from meta keys (version, timestamp, frame_count)
     - Return `MidlineFixture(frames=tuple(ordered_frames), frame_indices=tuple(sorted_indices), camera_ids=tuple(camera_ids), metadata=metadata)`
