    ## store_schema.py
    - SCHEMA_SQL string with CREATE TABLE IF NOT EXISTS for:
      - `samples` table: id (TEXT PK, UUID4), content_hash (TEXT NOT NULL), source (TEXT NOT NULL: manual/corrected/pseudo), image_path (TEXT NOT NULL, relative), label_path (TEXT NOT NULL, relative), parent_id (TEXT FK nullable, for augmented variants), import_batch_id (TEXT nullable), tags (TEXT DEFAULT '[]', JSON array), provenance (TEXT DEFAULT '[]', JSON array), metadata (TEXT DEFAULT '{}', JSON dict), created_at (TEXT NOT NULL), updated_at (TEXT NOT NULL), FOREIGN KEY parent_id REFERENCES samples(id) ON DELETE CASCADE
      - `datasets` table: name (TEXT PK), query_recipe (TEXT NOT NULL, JSON), sample_ids (TEXT NOT NULL, JSON array), split_seed (INTEGER NOT NULL), created_at (TEXT NOT NULL)
      - `models` table: run_id (TEXT PK), dataset_name (TEXT FK nullable), weights_path (TEXT NOT NULL), model_type (TEXT NOT NULL), metrics (TEXT DEFAULT '{}', JSON), tag (TEXT nullable), created_at (TEXT NOT NULL), FOREIGN KEY dataset_name REFERENCES datasets(name)
    - Indexes: idx_content_hash, idx_source, idx_parent_id, idx_import_batch_id
    - SOURCE_PRIORITY dict: {"pseudo": 0, "corrected": 1, "manual": 2}

    ## store.py — SampleStore class
    - `__init__(self, db_path: Path)`: Sets db_path, derives root (db_path.parent), images_dir (root/"images"), labels_dir (root/"labels"). Lazy connection.
    - `_connect(self) -> sqlite3.Connection`: Creates connection with WAL mode, foreign_keys=ON, Row factory, busy_timeout=5000. Runs SCHEMA_SQL on first connect.
    - `close(self)`: Closes connection if open.
    - `__enter__`/`__exit__`: Context manager support.
    - `compute_content_hash(path: Path) -> str`: Static method. SHA-256 of file bytes in 8192-byte chunks.
    - `import_sample(self, image_path: Path, label_path: Path, source: str, metadata: dict | None = None, import_batch_id: str | None = None) -> tuple[str, str]`: Returns (sample_id, action) where action is "imported", "upserted", or "skipped".
      - Computes content hash of image
      - Checks for existing sample with same hash
      - If exists and incoming source priority >= existing: cascade delete children (files + rows), replace label file, update DB row (source, label_path, metadata, provenance append "replaced"), return (existing_id, "upserted")
      - If exists and incoming source priority < existing: return (existing_id, "skipped")
      - If new: generate UUID4, copy image to images_dir/{uuid}.{ext}, copy label to labels_dir/{uuid}.txt, insert row with provenance [{"action": "imported", "source": source, "timestamp": iso}], return (uuid, "imported")
    - `_cascade_delete_children(self, parent_id: str) -> int`: Queries children by parent_id, deletes their files (images + labels), deletes DB rows. Returns count.
    - `add_augmented(self, parent_id: str, image_path: Path, label_path: Path, metadata: dict | None = None) -> str`: Imports an augmented variant with parent_id set. Source inherited from parent. Tags include "augmented".
    - `query(self, source: str | None = None, tags_include: list[str] | None = None, tags_exclude: list[str] | None = None, exclude_excluded: bool = True) -> list[dict]`: Returns matching samples as dicts. Uses json_each for tag filtering. exclude_excluded=True filters out samples with "excluded" tag.
    - `get(self, sample_id: str) -> dict | None`: Get single sample by ID.
    - `count(self, source: str | None = None) -> int`: Count samples matching filter.
    - `exclude(self, sample_ids: list[str]) -> int`: Add "excluded" tag to samples. Cascade: also exclude augmented children. Returns count modified.
    - `remove(self, sample_ids: list[str]) -> int`: Hard delete: delete files + DB rows. Children cascade via FK. Returns count deleted.
    - `_delete_files(self, sample_id: str, image_ext: str = ".jpg")`: Removes image and label files from disk.
    - `_append_provenance(self, existing_json: str, action: str, source: str, batch_id: str | None = None) -> str`: Parse JSON, append entry, return JSON string.
