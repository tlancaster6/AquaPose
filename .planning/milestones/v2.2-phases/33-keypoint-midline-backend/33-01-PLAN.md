    **1. Rewrite `src/aquapose/core/midline/backends/direct_pose.py`:**

    Replace the stub entirely with a working implementation. The class signature:

    ```python
    class DirectPoseBackend:
        def __init__(
            self,
            weights_path: str | Path,
            device: str = "cuda",
            n_points: int = 15,
            n_keypoints: int = 6,
            keypoint_t_values: list[float] | None = None,
            confidence_floor: float = 0.1,
            min_observed_keypoints: int = 3,
            crop_size: tuple[int, int] = (128, 128),
            **kwargs: Any,  # absorb unknown kwargs from get_backend
        ) -> None: ...

        def process_frame(
            self,
            frame_idx: int,
            frame_dets: dict[str, list[Detection]],
            frames: dict[str, np.ndarray],
            camera_ids: list[str],
        ) -> dict[str, list[AnnotatedDetection]]: ...
    ```

    **Constructor (`__init__`):**
    - Fail-fast: validate `weights_path` exists (same pattern as SegmentThenExtractBackend).
    - Store all config params as private attributes.
    - Default `keypoint_t_values`: `np.linspace(0.0, 1.0, n_keypoints).tolist()` — uniform spacing from nose to tail.
    - Lazy-import torch and `_PoseModel` from `aquapose.training.pose` at construction time (not at module level — avoids import issues when training/ is not fully set up).
    - Load model: `_PoseModel(n_keypoints=n_keypoints, pretrained=False)`, load state dict via `torch.load(weights_path, map_location="cpu", weights_only=True)`, move to device, set eval mode.
    - Store model as `self._model`.

    **`process_frame` method:**
    For each camera in camera_ids, for each detection in frame_dets[cam_id]:

    1. **Extract affine crop:** Use `extract_affine_crop` from `aquapose.segmentation.crop`.
       - `center_xy = det.centroid`
       - `angle_math_rad = det.angle if det.angle is not None else 0.0` (handle axis-aligned YOLO)
       - `obb_w, obb_h = det.bbox[2], det.bbox[3]` (width, height from bbox)
       - `crop_size = self._crop_size` (always (128, 128) for model input)
       - The model was trained at 128x128, so always crop at 128x128 regardless of DetectionConfig.crop_size.

    2. **Run inference:** `torch.no_grad()` context, convert crop image to tensor `(1, 3, 128, 128)` float [0,1], run through `self._model`, get `(1, n_keypoints*2)` sigmoid output, reshape to `(n_keypoints, 2)` in [0,1] crop-normalized coordinates.

    3. **Generate per-keypoint confidence:** Since `_PoseModel` has no explicit confidence head, use a heuristic: for each keypoint, confidence = `1.0 - 2.0 * max(abs(x - 0.5), abs(y - 0.5))`. This gives conf=1.0 for points at center (model confident), conf=0.0 for points at edges (model uncertain, pushing to boundary). This is a reasonable proxy since sigmoid outputs near 0 or 1 indicate the model is trying to place the point outside the crop, and near 0.5 means uncertain about position. Clip to [0, 1].

    4. **Apply confidence floor:** For each keypoint, if confidence < `self._conf_floor`, mark as not visible. Count visible keypoints. If visible count < `self._min_observed`, return an AnnotatedDetection with `midline=None` (empty midline, same as degenerate mask in segment-then-extract).

    5. **Back-project to frame space:** Convert visible keypoint normalized coords to crop pixel coords (multiply by 128), then use `invert_affine_points(kps_crop_px, affine.M)` to get frame-space coords.

    6. **Fit spline and resample:** Using `scipy.interpolate.CubicSpline`:
       - Get visible t-values from `self._t_values` for visible keypoints only.
       - Fit CubicSpline through visible (t, frame_xy) pairs. Use `extrapolate=False`.
       - Evaluate at `t_eval = np.linspace(0.0, 1.0, self._n_points)`.
       - NaN-pad: only fill points where `t_min_observed <= t_eval <= t_max_observed`. All other points get NaN.

    7. **Build confidence array:** Interpolate per-keypoint confidence along the spline using `scipy.interpolate.interp1d(visible_t, visible_conf, bounds_error=False, fill_value=0.0)`. Evaluate at t_eval. Zero out confidence where coordinates are NaN.

    8. **Build Midline2D:**
       ```python
       midline = Midline2D(
           points=midline_pts,           # (n_points, 2) float32
           half_widths=np.zeros(self._n_points, dtype=np.float32),  # no half-width info from keypoints
           fish_id=det.fish_id if hasattr(det, 'fish_id') else -1,
           camera_id=cam_id,
           frame_index=frame_idx,
           is_head_to_tail=True,  # keypoints are anatomically ordered nose->tail
           point_confidence=midline_conf,  # (n_points,) float32
       )
       ```

    9. **Build AnnotatedDetection:**
       ```python
       ad = AnnotatedDetection(
           detection=det,
           mask=None,           # no mask in keypoint backend
           crop_region=None,    # using affine crop, not axis-aligned CropRegion
           midline=midline,
       )
       ```

    Return dict mapping camera_id to list of AnnotatedDetections.
