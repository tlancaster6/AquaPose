  <action>
Create `scripts/detect_swaps.py` — a standalone script that validates swap detection on the YH dataset. Hardcode paths per the standalone-scripts-first workflow preference.

**Script structure:**

1. **Configuration section** at top:
   - `RUN_DIR = Path("~/aquapose/projects/YH/runs/run_20260307_140127/").expanduser()` (Phase 72 baseline run)
   - `PROJECTION_HEAD_PATH` — path to best_reid_model.pt from Phase 104 (check `scripts/train_reid_head.py` for the actual path used)
   - Known ground-truth swap events:
     - MF swap: fish 0 <-> 5, frame range ~2650-2680
     - FF swap: fish 2 <-> 4, frame range ~580-620
   - Clean fish IDs for FP measurement: all IDs not involved in known swaps

2. **Run seeded mode:**
   - Instantiate `SwapDetector(run_dir, config, projection_head_path)`
   - `seeded_events = detector.run(mode="seeded")`
   - Print summary: each event with frame, fish pair, margin, action

3. **Run scan mode:**
   - `scan_events = detector.run(mode="scan")`
   - Print summary: each event with frame, fish pair, margin, action, detection_mode

4. **Validation report:**
   - **Known MF swap detection**: Check if any confirmed/repaired event in seeded_events matches fish {0,5} pair within frame range 2600-2700. Report HIT/MISS.
   - **Known FF swap detection**: Check if any confirmed/repaired event in scan_events matches fish {2,4} pair within frame range 550-650. Report HIT/MISS.
   - **False positive rate**: Count events on clean fish IDs (those not involved in known swaps). FP rate = FP_count / total_clean_pairs_tested. Report rate and whether < 5%.
   - **Combined event list**: Merge seeded + scan events, deduplicate by (frame, fish_a, fish_b), prefer scan mode if both detect same event.

5. **Apply repair:**
   - Filter confirmed events (action="confirmed" -> set action="repaired")
   - Call `detector.repair(repaired_events)`
   - Print output path

6. **Console summary** (printed at end):
   ```
   === Swap Detection Validation Report ===
   Seeded mode: N events (M confirmed, K rejected)
   Scan mode: N events (M confirmed, K rejected, J skipped)

   Known swap detection:
     MF (fish 0<->5, ~frame 2665): HIT/MISS (margin=X.XXX)
     FF (fish 2<->4, ~frame 600):  HIT/MISS (margin=X.XXX)

   False positive rate: X/Y = Z.Z% (PASS/FAIL, threshold 5%)

   Output: {path_to_midlines_reid.h5}
   ```

**Important:**
- Use `.cpu().numpy()` for any CUDA tensor conversions
- The script should be runnable with `hatch run python scripts/detect_swaps.py`
- If either known swap is MISSED, print diagnostic info (all embeddings found for that fish pair in the relevant window, cosine similarities)
- Handle the case where the projection head file doesn't exist gracefully (fall back to raw 768-dim embeddings with a warning)
