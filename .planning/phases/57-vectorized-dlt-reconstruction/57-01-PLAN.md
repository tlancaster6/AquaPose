<objective>
Replace the per-body-point Python loop in `DltBackend._reconstruct_fish()` with a single vectorized method `_triangulate_fish_vectorized()` that batches all N body points through ray casting, normal-equation assembly, lstsq solve, reprojection, outlier rejection, and re-triangulation. The scalar `_triangulate_body_point()` method is retained as an unreachable reference.

Purpose: Eliminates the N-body-point Python dispatch overhead (reconstruction is ~9% of wall time). Ray casting goes from N*C calls to C calls, and lstsq solves N systems in one batched call.

Output: Updated `dlt.py` with vectorized triangulation path; equivalence tests confirming correctness.
</objective>

    Add `_triangulate_fish_vectorized(self, cam_midlines: dict[str, Midline2D], water_z: float) -> _TriangulationResult` as a private method on DltBackend. Implementation follows the research patterns:

    **Step 1 — Build (N, C, 2) pixel tensor and (N, C) validity/weight masks:**
    - `cam_ids = [cid for cid in cam_midlines if cid in self._models]` (ordered list, length C)
    - Stack all N body points for all C cameras into `pixels_nc` shape (N, C, 2)
    - `valid_nc` shape (N, C) bool: True where pixel is not NaN
    - `weights_nc` shape (N, C) float: `sqrt(confidence)` where valid, 0.0 where invalid
    - Replace NaN pixels with 0.0 in `pixels_safe` (zero-weight handles them)

    **Step 2 — Cast rays: C calls, each processing N body points:**
    ```python
    origins_nc = torch.zeros(N, C, 3, device=device, dtype=dtype)
    dirs_nc = torch.zeros(N, C, 3, device=device, dtype=dtype)
    for c, cam_id in enumerate(cam_ids):
        o, d = self._models[cam_id].cast_ray(pixels_safe[:, c, :])  # (N, 3) each
        origins_nc[:, c, :] = o
        dirs_nc[:, c, :] = d
    ```

    **Step 3 — First-pass normal-equation assembly and lstsq solve:**
    Loop over C cameras (NOT N body points) accumulating into (N, 3, 3) A and (N, 3) b:
    ```python
    eye3 = torch.eye(3, device=device, dtype=dtype)
    A = torch.zeros(N, 3, 3, device=device, dtype=dtype)
    b = torch.zeros(N, 3, device=device, dtype=dtype)
    for c in range(C):
        d = dirs_nc[:, c, :]     # (N, 3)
        o = origins_nc[:, c, :]  # (N, 3)
        w = weights_nc[:, c]     # (N,)
        ddt = torch.einsum('ni,nj->nij', d, d)   # (N, 3, 3)
        M = eye3.unsqueeze(0) - ddt              # (N, 3, 3)
        wM = w[:, None, None] * M               # (N, 3, 3)
        A = A + wM
        b = b + torch.einsum('nij,nj->ni', wM, o)
    ```
    Pre-filter: body points with `valid_nc.sum(dim=1) < 2` get `has_enough = False`.
    Solve: `pts_3d = torch.linalg.lstsq(A, b.unsqueeze(-1)).solution.squeeze(-1)` for all N (invalid ones will be masked out).

    **Step 4 — Per-camera reprojection residuals (C calls, not N*C):**
    ```python
    residuals_nc = torch.full((N, C), float('inf'), device=device, dtype=dtype)
    for c, cam_id in enumerate(cam_ids):
        proj_px, proj_valid = self._models[cam_id].project(pts_3d)
        obs_px = pixels_nc[:, c, :]
        err = torch.linalg.norm(proj_px - obs_px, dim=-1)
        keep = proj_valid & valid_nc[:, c]
        residuals_nc[:, c] = torch.where(keep, err, torch.tensor(float('inf'), ...))
    ```

    **Step 5 — Build inlier masks and pre-filter for second pass:**
    ```python
    inlier_nc = residuals_nc <= self._outlier_threshold  # (N, C)
    inlier_count = inlier_nc.sum(dim=1)                  # (N,)
    has_enough_inliers = inlier_count >= 2               # (N,)
    weights_nc_inlier = weights_nc * inlier_nc.float()
    ```

    **Step 6 — Second-pass lstsq with inlier weights:**
    Re-run the A, b assembly loop using `weights_nc_inlier` instead of `weights_nc`. Solve again. Only valid for `has_enough_inliers` body points.

    **Step 7 — Water surface rejection (post re-triangulation only):**
    ```python
    above_water = pts_3d_pass2[:, 2] <= water_z
    valid_mask = has_enough_inliers & ~above_water
    ```

    **Step 8 — Compute mean inlier residuals for pass-2 points:**
    Re-project pass-2 pts_3d per camera, compute mean residual per body point among inlier cameras.

    **Step 9 — Build inlier_cam_ids from inlier_nc and cam_ids:**
    ```python
    inlier_cam_ids = [
        [cam_ids[c] for c in range(C) if inlier_nc[n, c].item()]
        for n in range(N)
    ]
    ```
