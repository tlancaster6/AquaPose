  <name>Task 1: Dataset assembly module with pooling, confidence filtering, and validation splits</name>
  <files>src/aquapose/training/dataset_assembly.py, tests/unit/training/test_dataset_assembly.py</files>
  <behavior>
    - collect_pseudo_labels(run_dirs, source, model_type) discovers and returns list of (image_path, label_path, confidence_entry) tuples from pseudo-label directories
    - filter_by_confidence(labels, min_confidence) removes labels below threshold
    - filter_by_gap_reason(labels, exclude_reasons) removes gap labels matching excluded reasons
    - split_manual_val(manual_dir, val_fraction, seed) splits manual images per-camera stratified, returns (train_paths, val_paths)
    - split_pseudo_val(labels, val_fraction, seed) holds out a fraction of pseudo-labels for evaluation
    - assemble_dataset(output_dir, ...) writes YOLO-format directory with images/{train,val}/, labels/{train,val}/, dataset.yaml
    - dataset.yaml val key points to manual val images (not pseudo-label val)
    - pseudo-label val metadata sidecar records source (consensus/gap), gap_reason, confidence per image
    - Multi-run image name collisions resolved by prefixing with run directory name
  </behavior>
  <action>
Create `src/aquapose/training/dataset_assembly.py` with:

1. `collect_pseudo_labels(run_dirs: list[Path], source: str, model_type: str) -> list[dict]`:
   - `source` is "consensus" or "gap"
   - `model_type` is "obb" or "pose"
   - For each run_dir, read `pseudo_labels/{source}/confidence.json`
   - Glob `pseudo_labels/{source}/{model_type}/images/train/*.jpg` and match with labels
   - Return list of dicts: `{"image_path": Path, "label_path": Path, "stem": str, "confidence": float, "source": str, "run_id": str, "metadata": dict}`
   - `run_id` derived from run_dir.name for collision avoidance
   - Mean confidence across fish in that image used as image-level confidence

2. `filter_by_confidence(labels: list[dict], min_confidence: float) -> list[dict]`: Filter by image-level mean confidence.

3. `filter_by_gap_reason(labels: list[dict], exclude_reasons: list[str]) -> list[dict]`: For gap labels, exclude images where ALL fish have a gap_reason in the exclude list.

4. `split_manual_val(manual_dir: Path, val_fraction: float, seed: int) -> tuple[list[Path], list[Path]]`:
   - Read manual dataset.yaml to find images/labels structure
   - Extract camera_id from image filenames (pattern: `{frame}_{cam_id}.jpg`)
   - Per-camera stratified split: for each camera, hold out `val_fraction` images
   - Return (train_image_stems, val_image_stems)

5. `split_pseudo_val(labels: list[dict], val_fraction: float, seed: int) -> tuple[list[dict], list[dict]]`:
   - Random split of pseudo-labels into train and val portions
   - Return (train_labels, val_labels)

6. `assemble_dataset(output_dir: Path, manual_dir: Path | None, run_dirs: list[Path], model_type: str, consensus_threshold: float, gap_threshold: float, exclude_gap_reasons: list[str], manual_val_fraction: float, pseudo_val_fraction: float, seed: int) -> dict`:
   - Collect consensus labels from all run_dirs, filter by consensus_threshold
   - Collect gap labels from all run_dirs, filter by gap_threshold and exclude_gap_reasons
   - Split manual into train/val
   - Split pseudo-labels into train/val
   - Copy/symlink images and labels into output_dir/{images,labels}/{train,val}/
   - Prefix pseudo-label filenames with `{run_id}_` to avoid collisions
   - Manual val images go into `images/val/`, `labels/val/`
   - Write `dataset.yaml` with `val: images/val` (manual val only)
   - Write `pseudo_val_metadata.json` sidecar listing pseudo-label val images with source, gap_reason, confidence
   - Return summary dict with counts

Use shutil.copy2 for file copying (not symlinks -- portable across systems).

Write tests using tmp_path fixture with synthetic directory structures mimicking the pseudo-label output format. Test confidence filtering, gap-reason filtering, manual val splitting, and full assembly.

Add `assemble` command to `pseudo_label_group` in `pseudo_label_cli.py`:

```
@pseudo_label_group.command("assemble")
```

CLI options (following existing Click patterns):
- `--run-dir` (multiple=True, required=True, type=click.Path(exists=True)): One or more pipeline run directories containing pseudo_labels/
- `--manual-dir` (optional, type=click.Path(exists=True)): YOLO-format manual annotation directory
- `--output-dir` (required, type=click.Path()): Output directory for assembled dataset
- `--model-type` (type=click.Choice(["obb", "pose"]), required=True): Which label type to assemble
- `--consensus-threshold` (default=0.5, type=float): Min confidence for Source A labels
- `--gap-threshold` (default=0.3, type=float): Min confidence for Source B labels
- `--exclude-gap-reason` (multiple=True, type=str): Gap reasons to exclude (e.g., "no-tracklet")
- `--manual-val-fraction` (default=0.2, type=float): Fraction of manual data for validation
- `--pseudo-val-fraction` (default=0.1, type=float): Fraction of pseudo-labels held out for evaluation
- `--temporal-step` (default=1, type=int): Temporal subsampling step (1 = no subsampling)
- `--diversity-bins` (default=5, type=int): Number of curvature bins for diversity sampling
- `--diversity-max-per-bin` (default=None, type=int): Max frames per bin (None = no limit)
- `--seed` (default=42, type=int): Random seed

The command should:
1. If `--temporal-step > 1` or `--diversity-bins` specified: load diagnostic caches from each run_dir to get midlines_3d, apply temporal_subsample and diversity_sample, then filter pseudo-labels to only include selected frames
2. Call `assemble_dataset()` from dataset_assembly module
3. Print summary (counts per source, confidence stats, val split sizes)

Frame selection integration: The assemble command loads midlines_3d from diagnostic caches (using `load_run_context` via dynamic import as in existing `generate` command) to perform frame selection. Selected frame indices are used to filter which pseudo-labels are included. If `--temporal-step 1` and no diversity options, skip frame selection entirely (no diagnostic cache loading needed).

Update `__init__.py` to export `assemble_dataset` from dataset_assembly.

Add a basic CLI smoke test in `test_pseudo_label_cli.py` that invokes `assemble` with synthetic directories and verifies it produces output.
