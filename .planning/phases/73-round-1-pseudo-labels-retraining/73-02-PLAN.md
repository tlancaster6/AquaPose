aquapose --project ~/aquapose/projects/YH data import \

<task type="auto">
  <name>Task 2: Import CVAT corrections, quantify, and assemble curated datasets</name>
  <files></files>
  <action>
**Step 1: Quantify correction magnitude.**

Before importing, compare original pseudo-labels against CVAT-corrected exports:
- **OBB**: For each image in the CVAT export, load original pseudo-label and corrected label. Compute per-box IoU between matched boxes (match by highest IoU). Report: number of images modified, mean IoU between original and corrected boxes, count of boxes added/removed.
- **Pose**: For each crop, compute per-keypoint Euclidean displacement (pixels) between original and corrected positions. Report: number of crops modified, mean/median/p90 keypoint displacement, count of visibility flag changes, count of crops removed.
- **Summary**: Total samples reviewed, % modified, % removed, mean correction magnitude. Save report to `~/aquapose/projects/YH/training_data/round1_selected/correction_report.json`.

**Step 2: Import CVAT-corrected labels as source=manual.**

```bash
aquapose --project ~/aquapose/projects/YH data import \
    --store obb --source manual \
    --input-dir {CVAT_OBB_EXPORT_DIR} \
    --metadata-json '{"round": 1, "corrected_from": "pseudo"}' \
    --batch-id "round1-obb-corrected"

aquapose --project ~/aquapose/projects/YH data import \
    --store pose --source manual \
    --input-dir {CVAT_POSE_EXPORT_DIR} \
    --metadata-json '{"round": 1, "corrected_from": "pseudo"}' \
    --batch-id "round1-pose-corrected"
```

Source=manual ensures corrected labels take priority over original pseudo versions during assembly.

**Step 3: Exclude samples the user removed during CVAT curation.**

For any pseudo-label samples that the user deleted in CVAT (present in original selection but absent from CVAT export):
```bash
aquapose --project ~/aquapose/projects/YH data exclude \
    --store {obb|pose} --ids {image_stem} --reason removed-in-curation --by-filename
```

**Step 4: Assemble curated datasets.**

```bash
aquapose --project ~/aquapose/projects/YH data assemble \
    --store obb --name round1-curated --split-mode tagged
aquapose --project ~/aquapose/projects/YH data assemble \
    --store pose --name round1-curated --split-mode tagged
```

Verify pose dataset.yaml contains `kpt_shape` and `flip_idx`.

**Step 5: Verify dataset composition.**
- Uncurated: manual + original pseudo-labels (assembled before corrections)
- Curated: manual + corrected labels (source=manual overrides pseudo) minus excluded samples
- Print counts for all 4 datasets to confirm differences match correction summary
