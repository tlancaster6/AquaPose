   ```python
   class FishEmbedder:
       """Stateful GPU wrapper around MegaDescriptor-T for fish re-identification.

       Loads the backbone once at construction time, holds it on GPU.
       Accepts BGR uint8 crops of any size, preprocesses internally
       (resize to crop_size x crop_size, normalize to [-1, 1]),
       and returns L2-normalized embedding vectors.

       Args:
           config: ReidConfig with model_name, batch_size, crop_size, device, embedding_dim.
       """
   ```

   Constructor (`__init__(self, config: ReidConfig)`):
   - Load model via `timm.create_model(config.model_name, num_classes=0, pretrained=True)`
   - Move to `config.device`, set to `.eval()`
   - Store config fields as private attrs (`_device`, `_batch_size`, `_crop_size`, `_embedding_dim`)
   - Verify output dim: run a dummy forward pass with `torch.randn(1, 3, crop_size, crop_size)` and assert output shape is `(1, embedding_dim)`. Raise `ValueError` if mismatch.

   `embed_batch(self, crops: list[np.ndarray]) -> np.ndarray`:
   - Accept list of BGR uint8 numpy arrays (any size)
   - Preprocess each crop: BGRŌåÆRGB via `cv2.cvtColor`, resize to `(crop_size, crop_size)` via `cv2.resize`, convert to float32, divide by 255.0, normalize `(x - 0.5) / 0.5` to get [-1, 1] range (per MegaDescriptor model card ŌĆö NOT ImageNet stats)
   - Stack into a `(N, 3, H, W)` tensor, move to device
   - Process in sub-batches of `_batch_size` with `torch.no_grad()` context
   - L2-normalize via `F.normalize(feats, p=2, dim=1)`
   - Return as numpy float32 array on CPU (use `.cpu().numpy()`, NEVER bare `.numpy()`)
   - Handle empty input: return empty `(0, embedding_dim)` array

   Type hints on all public methods. Google-style docstrings.
  </action>
