  <behavior>
    - temporal_subsample(frame_indices, step=K) returns every Kth frame index from sorted input
    - temporal_subsample with step=1 returns all frames unchanged
    - filter_empty_frames(frame_indices, midlines_3d) removes frames where midlines_3d[frame_idx] is empty dict or missing
    - filter_empty_frames with all-empty input returns empty list
    - compute_curvature(control_points) returns a scalar curvature value from (7,3) control points using finite differences of the tangent vectors (mean absolute curvature along the spline)
    - compute_curvature on a straight line returns ~0
    - compute_curvature on a curved line returns > 0
    - diversity_sample(midlines_3d, frame_indices, n_bins=5, max_per_bin=None, seed=42) clusters per-fish curvatures into K-means bins and samples equally from each bin; returns selected frame indices (union across all fish)
    - diversity_sample with max_per_bin=1 returns at most n_bins frames per fish
    - diversity_sample preserves all frames if max_per_bin is None (returns all input frames, just validates they have reconstructions)
  </behavior>
  <action>
Create `src/aquapose/training/frame_selection.py` with these functions:

1. `temporal_subsample(frame_indices: list[int], step: int) -> list[int]`: Sort input, return every `step`-th element. Simple slicing.

2. `filter_empty_frames(frame_indices: list[int], midlines_3d: list[dict]) -> list[int]`: Keep only frame indices where `midlines_3d[idx]` exists and is non-empty dict.

3. `compute_curvature(control_points: np.ndarray) -> float`: Compute mean absolute curvature from (7,3) control points. Use finite differences: tangent vectors T[i] = cp[i+1] - cp[i], then curvature k[i] = |T[i+1] - T[i]| / (0.5*(|T[i]| + |T[i+1]|)). Return mean of k values. This captures bending without needing scipy.

4. `diversity_sample(midlines_3d: list[dict], frame_indices: list[int], n_bins: int = 5, max_per_bin: int | None = None, seed: int = 42) -> list[int]`:
   - For each fish across all candidate frames, compute curvature from control_points
   - Collect all (fish_id, frame_idx, curvature) triples
   - Use scipy.cluster.vq.kmeans2 (or sklearn KMeans -- use scipy to avoid heavy sklearn dependency) to assign curvature values to `n_bins` clusters
   - For each bin, if `max_per_bin` is not None, randomly sample `max_per_bin` frames (seeded); otherwise keep all
   - Return sorted union of selected frame indices across all fish

Update `src/aquapose/training/__init__.py` to export `temporal_subsample`, `filter_empty_frames`, `compute_curvature`, `diversity_sample`.

Write tests in `tests/unit/training/test_frame_selection.py` covering all behaviors listed above, using synthetic Midline3D objects (straight-line and curved control points).
