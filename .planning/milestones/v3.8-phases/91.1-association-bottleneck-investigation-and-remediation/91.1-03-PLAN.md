<objective>
Vectorize all scalar ray-casting in recovery.py and restructure _attempt_split_assign to precompute residuals before sweeping split points.

Purpose: Recovery takes ~437s per chunk (97% of association time). The root cause is _attempt_split_assign calling _score_singleton_against_group (which itself has nested scalar loops) for every (split_point x group) combination. Precomputing a (n_groups, n_frames) residual matrix and sweeping with numpy reduces this from O(splits x groups x frames x keypoints) Python iterations to O(groups x frames x keypoints) batch ops + O(splits x groups) numpy slicing.
Output: Vectorized recovery.py, all existing tests passing.

**Vectorize `_triangulate_group_keypoints_for_frame`:**

Current code: for each keypoint k, loops over group tracklets casting rays one pixel at a time, then pairwise scalar `ray_ray_closest_point`. Restructure:

1. For each keypoint k, collect all tracklets that have a confident keypoint k in this frame
2. Batch all those pixels by camera: for each camera, call `cast_ray` once with all its keypoints for all keypoints that need rays from that camera
3. Actually, since this function is called per-frame and there are typically 3-6 cameras with 6 keypoints each, the total batch size is ~18-36 rays. Group by camera, batch cast_ray per camera:

```python

for t, fm in zip(group_tracklets, tracklet_frame_maps, strict=True):
    if frame not in fm or t.keypoints is None or t.keypoint_conf is None:
        continue
    if t.camera_id not in forward_luts:
        continue
    idx = fm[frame]
    for k in range(n_keypoints):
        if t.keypoint_conf[idx, k] < keypoint_confidence_floor:
            continue
        cam_kpt_pixels.setdefault(t.camera_id, []).append(
            (k, t.keypoints[idx, k])
        )

# Step 2: Batch cast_ray per camera
cam_rays: dict[str, list[tuple[int, np.ndarray, np.ndarray]]] = {}  # cam -> [(kpt_idx, origin, dir), ...]
for cam_id, kpt_pixels in cam_kpt_pixels.items():
    pixels = np.array([p for _, p in kpt_pixels], dtype=np.float64)
    pix_t = torch.tensor(pixels, dtype=torch.float32)
    origins, dirs = forward_luts[cam_id].cast_ray(pix_t)
    origins_np = origins.cpu().numpy().astype(np.float64)
    dirs_np = dirs.cpu().numpy().astype(np.float64)
    cam_rays[cam_id] = [
        (kpt_pixels[i][0], origins_np[i], dirs_np[i])
        for i in range(len(kpt_pixels))
    ]
```

3. For pairwise midpoints per keypoint: group rays by keypoint index, then for each keypoint with 2+ rays, compute pairwise midpoints. The pairwise loop is small (C(cameras, 2) per keypoint) so scalar `ray_ray_closest_point` is acceptable here since we need midpoints.

**Vectorize `_score_singleton_against_group`:**

This is the most impactful function. Current code: loops over shared frames, calling `_triangulate_group_keypoints_for_frame` per frame, then loops over keypoints casting singleton rays one-at-a-time.

Restructure the keypoint path:
1. Batch the singleton's cast_ray calls: collect ALL confident keypoint pixels across ALL shared frames into one flat array, make ONE `cast_ray` call for the singleton camera
2. Call `_triangulate_group_keypoints_for_frame` per frame (already vectorized above)
3. For each singleton ray, compute `_point_to_ray_distance` against the triangulated 3D point

For step 3, vectorize `_point_to_ray_distance` by adding a batch version:
```python
def _point_to_ray_distance_batch(
    points: np.ndarray,    # (N, 3)
    origins: np.ndarray,   # (N, 3)
    directions: np.ndarray, # (N, 3)
) -> np.ndarray:           # (N,)
    """Batch perpendicular distance from N 3D points to N rays."""
    w = points - origins  # (N, 3)
    t = np.sum(w * directions, axis=1, keepdims=True)  # (N, 1)
    closest = origins + t * directions  # (N, 3)
    return np.linalg.norm(points - closest, axis=1)  # (N,)
```

1. Check camera overlap per-frame (not just per-segment). Build a boolean mask `(n_groups, n_frames)` indicating which (group, frame) combinations are valid.
2. Call the vectorized `_score_singleton_against_group` to get per-frame residuals instead of a single mean. This requires modifying or creating a variant that returns per-frame residuals instead of a scalar mean.

**Actually, the cleaner approach**: Create a `_compute_per_frame_residuals` function that returns per-frame residuals (the same computation as `_score_singleton_against_group` but returning the array instead of the mean):

```python
def _compute_per_frame_residuals(
    singleton: Tracklet2D,
    group: TrackletGroup,
    forward_luts: dict[str, ForwardLUT],
    config: RecoveryConfigLike,
) -> np.ndarray | None:
    """Compute per-frame residuals for singleton vs group.

    Returns:
        Array of shape (n_singleton_frames,) with per-frame mean residuals,
        or np.nan for frames without valid data. Returns None if no shared
        frames exceed the minimum.
    """
```

This function:
- Finds shared frames between singleton and group
- Batches singleton cast_ray once (same as vectorized _score_singleton_against_group)
- Triangulates group 3D points per shared frame
- Returns per-frame residuals indexed by singleton frame index (not shared frame index)
- Uses np.nan for singleton frames not shared with the group

Then `_score_singleton_against_group` becomes a thin wrapper: `np.nanmean(per_frame_residuals)`.

**Step 2: Restructure `_attempt_split_assign`:**

```python
def _attempt_split_assign(singleton, multi_groups, existing_assignments, forward_luts, config, next_track_id):
    n_frames = len(singleton.frames)
    min_seg = config.recovery_min_segment_length
    if n_frames < 2 * min_seg:
        return None

    # Phase 1: Precompute per-frame residuals for all groups
    n_groups = len(multi_groups)
    residuals = np.full((n_groups, n_frames), np.nan)

    for g_idx, group in enumerate(multi_groups):
        per_frame = _compute_per_frame_residuals(singleton, group, forward_luts, config)
        if per_frame is not None:
            residuals[g_idx] = per_frame

    # Phase 2: Precompute camera overlap validity per split
    # For each group, determine the full-segment overlap (conservative: if ANY
    # detected frame overlaps, the whole segment is blocked)
    # We need per-frame detected-frame sets to evaluate per-segment overlap

    # Phase 3: Sweep split points
    best = None  # (total_res, split_idx, before_g_idx, after_g_idx)

    for split_idx in range(min_seg, n_frames - min_seg + 1):
        # Mean residual for before segment [:split_idx] per group
        before_means = np.nanmean(residuals[:, :split_idx], axis=1)  # (n_groups,)
        after_means = np.nanmean(residuals[:, split_idx:], axis=1)   # (n_groups,)

        # Check camera overlap for each segment against each group
        # (must re-check since overlap depends on the frame range)
        before_slice = _slice_tracklet(singleton, 0, split_idx, 0)  # dummy id
        after_slice = _slice_tracklet(singleton, split_idx, n_frames, 0)

        for g_idx in range(n_groups):
            # Skip if before has overlap or poor score
            combined = list(multi_groups[g_idx].tracklets) + existing_assignments.get(g_idx, [])
            if _has_camera_overlap_with_list(before_slice, combined):
                before_means[g_idx] = np.inf
            if _has_camera_overlap_with_list(after_slice, combined):
                after_means[g_idx] = np.inf

        # Apply threshold
        before_valid = before_means < config.recovery_residual_threshold
        after_valid = after_means < config.recovery_residual_threshold

        if not np.any(before_valid) or not np.any(after_valid):
            continue

        # Find best before and after groups (must be DIFFERENT)
        best_before_g = int(np.argmin(np.where(before_valid, before_means, np.inf)))
        best_after_g = int(np.argmin(np.where(after_valid, after_means, np.inf)))

        if best_before_g == best_after_g:
            # Try second-best for one of them
            # ... (implement fallback: mask out the shared index and try again)
            after_means_masked = after_means.copy()
            after_means_masked[best_before_g] = np.inf
            if not np.any(after_means_masked < config.recovery_residual_threshold):
                before_means_masked = before_means.copy()
                before_means_masked[best_after_g] = np.inf
                if not np.any(before_means_masked < config.recovery_residual_threshold):
                    continue
                best_before_g = int(np.argmin(np.where(before_means_masked < config.recovery_residual_threshold, before_means_masked, np.inf)))
            else:
                best_after_g = int(np.argmin(np.where(after_means_masked < config.recovery_residual_threshold, after_means_masked, np.inf)))

        if best_before_g == best_after_g:
            continue

        total = before_means[best_before_g] + after_means[best_after_g]
        if best is None or total < best[0]:
            best = (total, split_idx, best_before_g, best_after_g)

    if best is None:
        return None

    _, split_idx, before_g_idx, after_g_idx = best
    before_seg = _slice_tracklet(singleton, 0, split_idx, next_track_id)
    after_seg = _slice_tracklet(singleton, split_idx, n_frames, next_track_id + 1)
    return (before_seg, before_g_idx, after_seg, after_g_idx), 2
```
