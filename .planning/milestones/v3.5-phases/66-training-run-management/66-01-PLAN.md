  <name>Task 1: Create run_manager module with run directory, config snapshot, summary, and provenance functions</name>
  <files>src/aquapose/training/run_manager.py, tests/unit/training/test_run_manager.py, src/aquapose/training/__init__.py</files>
  <behavior>
    - create_run_dir: Given a project config path and model_type ("obb"), creates directory at {project_dir}/training/{model_type}/run_{timestamp}/ and returns the Path
    - create_run_dir: Raises ValueError if project_dir not set in config
    - snapshot_config: Writes config.yaml to run_dir with CLI args, copies confidence.json and dataset.yaml from dataset_dir as dataset_confidence.json and dataset_dataset.yaml
    - parse_best_metrics: Parses Ultralytics results.csv (with whitespace-stripped headers), returns dict with best_epoch (by mAP50-95), mAP50, mAP50-95, precision, recall, total_time
    - parse_best_metrics: Returns empty dict for empty CSV
    - write_summary: Creates summary.json with run_id, tag, model_type, model_variant, parent_weights, dataset_path, dataset_sources, training_config, metrics, training_duration_seconds, created timestamp
    - extract_dataset_provenance: Reads confidence.json and dataset.yaml from dataset dir to build source breakdown dict
    - resolve_project_dir: Extracts project_dir from YAML config, raises ValueError if missing
  </behavior>
  <action>
Create `src/aquapose/training/run_manager.py` with the following functions:

1. `resolve_project_dir(config_path: Path) -> Path` — Read project config with `yaml.safe_load` (NOT engine imports), extract `project_dir`, expand/resolve, raise ValueError if missing.

2. `create_run_dir(project_config_path: Path, model_type: str) -> Path` — Call resolve_project_dir, create `{project_dir}/training/{model_type}/run_{timestamp}/` with `datetime.now().strftime("%Y%m%d_%H%M%S")`, return path.

3. `snapshot_config(run_dir: Path, cli_args: dict, dataset_dir: Path | None = None) -> None` — Write cli_args as `config.yaml` via yaml.dump. If dataset_dir provided, copy `confidence.json` as `dataset_confidence.json` and `dataset.yaml` as `dataset_dataset.yaml` into run_dir (using shutil.copy2, skip if source missing).

4. `parse_best_metrics(results_csv: Path) -> dict` — Use csv.DictReader, strip whitespace from keys (Ultralytics quirk), find best row by `metrics/mAP50-95(B)`, return dict with best_epoch, mAP50, mAP50-95, precision, recall, total_time. Return empty dict if no rows.

5. `write_summary(run_dir: Path, results_csv_path: Path, training_args: dict, model_type: str, tag: str | None = None, dataset_dir: Path | None = None) -> None` — Call parse_best_metrics, call extract_dataset_provenance if dataset_dir provided, write pretty-printed summary.json (indent=2) with schema from RESEARCH.md.

6. `extract_dataset_provenance(dataset_dir: Path) -> dict` — Read confidence.json and dataset.yaml from dataset_dir, count images by source if pseudo_val_metadata.json exists, return provenance dict.

7. `print_next_steps(run_dir: Path, model_type: str, best_weights: Path) -> None` — Print training complete message with run_dir, best_weights path, and suggested compare/retrain commands using click.echo and click.style.

Use only stdlib + yaml + click (no engine imports). Follow Google-style docstrings, type hints on all public functions.

Create tests in `tests/unit/training/test_run_manager.py`:
