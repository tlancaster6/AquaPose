**1. Create `src/aquapose/core/detection/backends/yolo_obb.py`:**

Create a `YOLOOBBBackend` class following the same pattern as `YOLOBackend` in `yolo.py`:

- Constructor: `__init__(self, model_path, conf_threshold=0.5, iou_threshold=0.45, device="cuda")`.
  - Eagerly verify `model_path` exists (raise `FileNotFoundError` if not).
  - Lazily import `ultralytics.YOLO` and load the OBB model.

- `detect(self, frame: np.ndarray) -> list[Detection]`:
  - Run `model.predict(frame, conf=self._conf, iou=self._iou, verbose=False)`.
  - Read `result.obb` (NOT `result.boxes` — see Research pitfall 2).
  - Guard `if r.obb is None: continue`.
  - For each detection in `r.obb`:
    - Extract `xywhr` (cx, cy, w, h, angle_cw_rad) and `xyxyxyxy` (4 corners).
    - Use `.cpu().numpy()` for CUDA tensor safety.
    - **Angle conversion** (CRITICAL): `detection_angle = -float(angle_cw_rad)` (negate: ultralytics clockwise -> standard math CCW). This is the ONE place angle conversion happens.
    - Compute AABB fallback bbox from OBB corners: `corners.min(axis=0)` / `.max(axis=0)` -> `(x_min, y_min, w, h)` as ints.
    - Create `Detection(bbox=aabb, mask=None, area=int(w*h), confidence=conf, angle=detection_angle, obb_points=corners)`.
  - Return the list of detections. No padding_fraction here — padding is handled by the affine crop utility downstream.

Include a module-level docstring and `__all__`.

**1. Add affine crop utilities to `segmentation/crop.py`:**

Add these new items to the existing file (do NOT modify existing `CropRegion`, `compute_crop_region`, `extract_crop`, `paste_mask`):

- `AffineCrop` dataclass:
  ```python
  @dataclass
  class AffineCrop:
      image: np.ndarray       # (crop_h, crop_w, C) or (crop_h, crop_w)
      M: np.ndarray           # shape (2, 3), float64 — frame->crop transform
      crop_size: tuple[int, int]   # (width, height)
      frame_shape: tuple[int, int] # (height, width) of source frame
  ```

- `extract_affine_crop()` function:
  ```python
  def extract_affine_crop(
      frame: np.ndarray,
      center_xy: tuple[float, float],
      angle_math_rad: float,
      obb_w: float,
      obb_h: float,
      crop_size: tuple[int, int],
      padding_fraction: float = 0.15,
      interpolation: int = cv2.INTER_LINEAR,
  ) -> AffineCrop:
  ```
  Implementation:
  - Convert standard math angle to cv2 degrees: `angle_cv2_deg = math.degrees(-angle_math_rad)` (negate: math CCW -> cv2 CW).
  - Build rotation matrix: `M_rot = cv2.getRotationMatrix2D((cx, cy), angle_cv2_deg, scale=1.0)`.
  - Add translation to center OBB center in crop canvas: `M_rot[0,2] += crop_w/2 - cx; M_rot[1,2] += crop_h/2 - cy`.
  - Apply `cv2.warpAffine(frame, M_rot, (crop_w, crop_h), flags=interpolation, borderMode=cv2.BORDER_CONSTANT, borderValue=0)`.
  - `BORDER_CONSTANT` with value 0 implicitly handles letterbox padding (zero fills outside source).
  - Return `AffineCrop(image=crop_image, M=M_rot, crop_size=crop_size, frame_shape=frame.shape[:2])`.
  - Add `import math` at the top of the file.

- `invert_affine_point()` function:
  ```python
  def invert_affine_point(
      crop_xy: tuple[float, float],
      M: np.ndarray,
  ) -> tuple[float, float]:
  ```
  Implementation:
  - `M_inv = cv2.invertAffineTransform(M)` — numerically exact for rotation+translation.
  - `pt = np.array([[[crop_xy[0], crop_xy[1]]]], dtype=np.float64)`.
  - `result = cv2.transform(pt, M_inv)`.
  - Return `(float(result[0,0,0]), float(result[0,0,1]))`.

- `invert_affine_points()` (batch version for efficiency):
  ```python
  def invert_affine_points(
      crop_points: np.ndarray,
      M: np.ndarray,
  ) -> np.ndarray:
  ```
  - Takes points array of shape `(N, 2)` and returns `(N, 2)` in frame coordinates.
  - `M_inv = cv2.invertAffineTransform(M)`.
  - `pts = crop_points.reshape(1, -1, 2).astype(np.float64)`.
  - `result = cv2.transform(pts, M_inv)`.
  - Return `result.reshape(-1, 2)`.

Update `__all__` or add the new names if the module doesn't use `__all__` (it doesn't currently — no `__all__`).
