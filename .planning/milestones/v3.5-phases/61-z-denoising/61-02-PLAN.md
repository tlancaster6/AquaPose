    2. Create `src/aquapose/core/reconstruction/temporal_smoothing.py` with:

       `smooth_plane_normals(normals: np.ndarray, is_degenerate: np.ndarray, fish_ids: np.ndarray, frame_indices: np.ndarray, sigma_frames: int = 3) -> np.ndarray`
       - normals shape (T, 3), is_degenerate shape (T,) bool, fish_ids shape (T,) int, frame_indices shape (T,) int
       - These arrays represent a time series for a SINGLE fish (caller groups by fish_id)
       - Step 1: Sign consistency -- flip normals so dot(n_t, n_{t-1}) > 0 (iterate forward)
       - Step 2: Identify continuous segments -- frame gaps > 1 create segment boundaries
       - Step 3: For each segment, apply scipy.ndimage.gaussian_filter1d with sigma=sigma_frames to each normal component (x, y, z independently)
       - Step 4: Exclude degenerate frames from the filter input (replace with NaN, use a NaN-aware approach or interpolate first)
       - Step 5: Re-normalize each smoothed normal to unit length
       - Return smoothed normals shape (T, 3)

       `rotate_control_points_to_plane(control_points: np.ndarray, centroid: np.ndarray, raw_normal: np.ndarray, smoothed_normal: np.ndarray) -> np.ndarray`
       - control_points shape (7, 3), centroid shape (3,), raw/smoothed_normal shape (3,)
       - Compute rotation matrix R that maps raw_normal to smoothed_normal (Rodrigues' rotation formula)
       - Translate to centroid, rotate, translate back: `cp_new = R @ (control_points - centroid) + centroid`
       - Handle edge case: if raw_normal and smoothed_normal are nearly identical (dot > 0.9999), return original control points
       - Handle edge case: if raw_normal and smoothed_normal are nearly opposite (dot < -0.9999), use 180-degree rotation around an arbitrary perpendicular axis
       - Return rotated control_points shape (7, 3)

       ```python
       @dataclass(frozen=True)
       class ZDenoisingMetrics:
           median_z_range_cm: float          # Median z-range across fish (cm)
           mean_z_profile_rms_cm: float      # Mean frame-to-frame z-profile RMS (cm)
           per_fish_snr: dict[int, float]    # SNR per fish
           fish_above_snr_1: int             # Count of fish with SNR > 1

           residual_delta_px: float | None   # Component A gate: mean residual change (needs before/after)
       ```

       Implementation:
       - For z-range: evaluate spline at n_sample_points, compute max(z) - min(z) per fish per frame, then median across frames per fish, then median across fish
       - For z-profile RMS: compute z-profile (z values at sample points) per frame, diff consecutive frames for same fish, compute RMS of the diff
       - For SNR: signal = std of mean z-profile across time, noise = std of frame-to-frame z-profile changes
       - `residual_delta_px` is optional -- computed only when before/after residuals are available (e.g., from eval harness A/B comparison)
