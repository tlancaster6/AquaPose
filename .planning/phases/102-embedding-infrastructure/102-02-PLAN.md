**Step 2: Discover and iterate chunk caches**
- Glob `run_dir / "diagnostics" / "chunk_*" / "cache.pkl"`, sort by chunk number
- For each chunk cache:
  - Load via `load_chunk_cache(path)`
  - Determine `chunk_start` from the cache's chunk manifest. The cache path encodes chunk number (e.g., `chunk_000`). Use the pipeline config's `chunk_size` to compute: `chunk_start = chunk_number * chunk_size`. Read `chunk_size` from `run_dir / "config.yaml"` if available, or infer from the chunk cache's `frame_count` and chunk index.
  - **IMPORTANT**: Check for a `chunk_start` attribute on the loaded context or infer from chunk manifest. Look for `run_dir / "diagnostics" / "manifest.json"` which may contain chunk boundaries. If neither is available, compute from `chunk_number * frame_count` where `frame_count` is the cache's frame count.

- From `ctx.tracklet_groups`, build a lookup: `{(local_frame, camera_id): (fish_id, detection)}` by iterating each `TrackletGroup`'s `Tracklet2D` tracklets:
  - For each tracklet in `group.tracklets`:
    - For each index `i`, `local_frame = tracklet.frames[i]`:
      - Skip if `tracklet.frame_status[i] == "coasted"` (no real detection)
      - `cam_id = tracklet.camera_id`
      - `centroid = tracklet.centroids[i]` — `(u, v)`
      - Find the matching `Detection` in `ctx.detections[local_frame][cam_id]` by closest centroid distance (compute detection centroid as `(bbox[0] + bbox[2]/2, bbox[1] + bbox[3]/2)`)

      - Only process frames where `global_frame` is in `valid_frames`

**Step 4: Open video and extract crops**
- Use `VideoFrameSource` as context manager to open all camera videos
- Iterate frames sequentially (sorted by global frame index within each chunk) to match video seek pattern
- For each `(global_frame, cam_id)` with a matched detection:
  - Read frame via `frame_source.read_frame(global_frame)` → `{cam_id: bgr_frame}`
  - Extract crop: compute center from detection bbox `(x + w/2, y + h/2)`, use `detection.angle` (default 0.0 if None), use `detection.bbox[2]` as obb_w and `detection.bbox[3]` as obb_h

  - The crop_size is square for MegaDescriptor input (224x224 by default)
  - Collect crop images for batching
  - If `save_crops` is True, save crops to `run_dir / "reid" / "crops" / f"frame{global_frame:06d}_{cam_id}_fish{fish_id}.jpg"`

**Step 5: Batch embed and accumulate**
- Create `FishEmbedder(config)`
- Process collected crops through `embedder.embed_batch()` (the embedder handles internal sub-batching)
- Accumulate results into lists: `all_embeddings`, `all_frame_indices`, `all_fish_ids`, `all_camera_ids`, `all_confidences`
- For memory efficiency, process one chunk at a time: extract all crops for a chunk, embed them, append to accumulator lists, then proceed to next chunk

**Step 6: Write NPZ**
- Create `run_dir / "reid"` directory
- Stack all accumulated results into numpy arrays
- Save via `np.savez()`:
  - `embeddings`: `(N, 768)` float32
  - `frame_index`: `(N,)` int64
  - `fish_id`: `(N,)` int32
  - `camera_id`: `(N,)` object (string array)

**Progress reporting**: Print chunk progress to stdout: `"Embedding chunk {i+1}/{n_chunks} ({n_crops} crops)..."`. Print total counts at end.

**Error handling**:

- Log warnings for frames with no matching detection (should be rare)

1. Create `src/aquapose/core/reid/eval.py` with two public functions:

**`compute_reid_metrics(npz_path: Path, n_eval_frames: int = 100, seed: int = 42) -> dict[str, float]`**

Loads `embeddings.npz`, samples `n_eval_frames` unique frame indices uniformly at random (using `rng = np.random.default_rng(seed)`), collects all embeddings for those frames, and computes:

1. **Pairwise cosine similarity matrix** (dot product of L2-normalized embeddings = cosine sim):
   - Select all embeddings where `frame_index` is in the sampled frames
   - Compute `sim_matrix = embeddings @ embeddings.T`
   - Build boolean mask: `same_id[i, j] = (fish_ids[i] == fish_ids[j])` excluding diagonal
   - `mean_within = sim_matrix[same_id & ~diagonal].mean()`
   - `mean_between = sim_matrix[~same_id & ~diagonal].mean()`

2. **Rank-1 accuracy**:
   - For each query embedding, find the highest-similarity non-self embedding
   - Check if it has the same fish_id
   - `rank1 = n_correct / n_total`
   - IMPORTANT: exclude same-frame same-camera matches (self-retrieval). Two embeddings from different cameras of the same frame are valid matches.

3. **mAP (mean Average Precision)**:
   - For each query, rank all gallery items by cosine similarity (descending)
   - Compute AP as the mean of precisions at each relevant (same fish_id) position
   - `mAP = mean(AP_i for all queries)`

Return dict: `{"mean_within_sim": float, "mean_between_sim": float, "sim_gap": float, "rank1": float, "mAP": float, "n_eval_embeddings": int, "n_eval_frames": int}`

**`print_reid_report(metrics: dict[str, float]) -> None`**
