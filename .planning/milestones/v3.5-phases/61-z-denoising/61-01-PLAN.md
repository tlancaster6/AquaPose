  <action>
    1. Create `src/aquapose/core/reconstruction/plane_fit.py` with two public functions:

       `fit_plane_weighted(pts_3d: np.ndarray, weights: np.ndarray) -> tuple[np.ndarray, np.ndarray, bool]`
       - pts_3d shape (M, 3), weights shape (M,) -- weights are per-point camera counts (as floats)
       - Compute weighted centroid: centroid = sum(w_i * p_i) / sum(w_i)
       - Center points: centered = pts_3d - centroid
       - Weight rows: weighted_centered = centered * sqrt(weights)[:, None]
       - SVD: U, S, Vt = np.linalg.svd(weighted_centered, full_matrices=False)
       - Normal = Vt[2] (row corresponding to smallest singular value)
       - Degenerate detection: if S[1] / S[0] < threshold (use 0.01 as threshold per Claude's discretion), set is_degenerate=True and normal=[0, 0, 1]
       - Return (normal, centroid, is_degenerate)

       `project_onto_plane(pts_3d: np.ndarray, normal: np.ndarray, centroid: np.ndarray) -> tuple[np.ndarray, np.ndarray]`
       - Compute signed off-plane distances: d = (pts_3d - centroid) @ normal
       - Project: pts_projected = pts_3d - d[:, None] * normal[None, :]
       - Return (pts_projected, d)  -- d is signed off-plane residuals

    2. Extend `Midline3D` dataclass with new optional fields (all default None/False):
       - `plane_normal: np.ndarray | None = None`  # shape (3,), float32

       - `off_plane_residuals: np.ndarray | None = None`  # shape (n_sample_points,), float32
       - `is_degenerate_plane: bool = False`

       Update the class docstring with the new fields.
