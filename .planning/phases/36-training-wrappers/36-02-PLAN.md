**`train_yolo_seg()` function signature:**
```python
def train_yolo_seg(
    data_dir: Path,
    output_dir: Path,
    *,
    epochs: int = 100,
    batch_size: int = 16,
    device: str | None = None,
    val_split: float = 0.2,
    imgsz: int = 640,
    model: str = "yolo26n-seg",
    weights: Path | None = None,
) -> Path:
```

**Core logic for both wrappers (seg and pose):**

1. **Read data.yaml** from `data_dir / "data.yaml"`. Parse YAML to get `train` and `val` NDJSON paths and any extra fields (for pose: `kpt_shape`, `kpt_names`, `flip_idx`).

2. **Convert NDJSON to YOLO .txt labels:**
   - For each split (train, val):
     - Read the NDJSON file referenced in data.yaml (e.g., `train.ndjson`)
     - Create `labels/{split}/` directory alongside existing `images/{split}/`
     - For each NDJSON record:
       - Extract image stem from `record["image"]`
       - Write `labels/{split}/{stem}.txt` with YOLO-format lines

   **Seg conversion** (one line per annotation):
   ```
   class_id x1 y1 x2 y2 ... xN yN
   ```
   Where coordinates are already normalized [0,1] in the NDJSON polygon field.

   **Pose conversion** (one line per annotation):
   ```
   class_id cx cy w h x1 y1 v1 x2 y2 v2 ... xN yN vN
   ```
   Where bbox and keypoints are already normalized [0,1] in the NDJSON. Visibility `v` values should be cast to int.

3. **Rewrite data.yaml** to a new file in `data_dir` (e.g., `data_ultralytics.yaml`):
   - `path:` set to `str(Path(data_dir).resolve())` (MUST be absolute — Ultralytics resolves relative paths against DATASETS_DIR)
   - `train: images/train`
   - `val: images/val`
   - Preserve `nc`, `names` from original
   - For pose: preserve `kpt_shape`, `kpt_names`, `flip_idx` from original data.yaml — read them, don't hardcode

4. **Initialize model:**
   - If `weights` is provided: `YOLO(str(weights))` (transfer learning)
   - Otherwise: `YOLO(f"{model}.pt")` (downloads pretrained if not cached)

5. **Call `model.train()`:**
   ```python
   results = model.train(
       data=str(rewritten_yaml_path),
       epochs=epochs,
       batch=batch_size,
       device=device,
       project=str(output_dir / "_ultralytics"),
       name="train",
       imgsz=imgsz,
   )
   ```

6. **Copy weights** to output_dir (same pattern as yolo_obb.py): `best_model.pt`, `last_model.pt`.

7. **MetricsLogger**: Log a single summary line with epochs and val_split (same as yolo_obb.py).

8. **Return** `Path` to best model weights.

**`seg` subcommand:**
```python
@train_group.command("seg")
@click.option("--data-dir", required=True, type=click.Path(exists=True), help="Directory with data.yaml and NDJSON seg dataset.")
@click.option("--output-dir", required=True, type=click.Path(), help="Directory for model weights and metrics.")
@click.option("--epochs", default=100, type=int, help="Number of training epochs.")
@click.option("--batch-size", default=16, type=int, help="Batch size.")
@click.option("--device", default=None, type=str, help="Torch device (auto-detect if omitted).")
@click.option("--val-split", default=0.2, type=float, help="Validation split fraction.")
@click.option("--imgsz", default=640, type=int, help="Training image size (square).")
@click.option("--model", default="yolo26n-seg", type=str, help="YOLO model variant (e.g. yolo26n-seg, yolo26s-seg).")
@click.option("--weights", default=None, type=click.Path(exists=True), help="Pretrained weights for transfer learning.")
def seg(...) -> None:
    """Train YOLO-seg instance segmentation model."""
    from .yolo_seg import train_yolo_seg
    best_path = train_yolo_seg(...)
    click.echo(f"Training complete. Best model: {best_path}")
```
