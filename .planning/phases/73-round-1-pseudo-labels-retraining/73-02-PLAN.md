**Step 1: Add `--include-excluded` flag to `assemble_cmd` in `data_cli.py`.**

Add a new click option before the `@click.pass_context` decorator:
```python
@click.option(
    "--include-excluded",
    is_flag=True,
    help="Include excluded samples (for uncurated A/B comparison).",
)
```

Update the function signature to accept `include_excluded: bool`. In the body, after building the `query` dict, add:
```python
if include_excluded:
    query["exclude_excluded"] = False
```

This passes through to `SampleStore.query(exclude_excluded=False)` via `assemble(**query)`.

**Step 2: Assemble 4 datasets.**

All assemblies use `--split-mode tagged` so that manual val samples (tagged "val" in Phase 71) and secondary pseudo-label val samples (tagged "val" during import in Plan 01) go to val.

OBB curated (excluded samples filtered out — default behavior):
```bash
aquapose --project ~/aquapose/projects/YH data assemble \
    --store obb --name round1-curated --split-mode tagged
```

OBB uncurated (include excluded samples):
```bash
aquapose --project ~/aquapose/projects/YH data assemble \
    --store obb --name round1-uncurated --split-mode tagged --include-excluded
```

Pose curated:
```bash
aquapose --project ~/aquapose/projects/YH data assemble \
    --store pose --name round1-curated --split-mode tagged
```

Pose uncurated:
```bash
aquapose --project ~/aquapose/projects/YH data assemble \
    --store pose --name round1-uncurated --split-mode tagged --include-excluded
```

**Step 3: Verify pose dataset.yaml files.**
Known bug: `store.assemble()` may not write `kpt_shape` and `flip_idx` in pose dataset.yaml. After each pose assembly, check the dataset.yaml and add if missing:
```yaml
kpt_shape: [6, 3]
flip_idx: [0, 1, 2, 3, 4, 5]
```

**Step 4: Verify all 4 datasets.**
Check train/val counts for each. Expected:
- Curated OBB: manual + non-excluded pseudo OBB in train, manual val in val
- Uncurated OBB: manual + ALL pseudo OBB in train, manual val in val
- Curated pose: manual (with augmented variants) + non-excluded pseudo pose in train, manual val + pseudo val in val
- Uncurated pose: manual (with augmented variants) + ALL pseudo pose in train, manual val + pseudo val in val

Print counts for all 4 datasets to confirm curated has fewer train samples than uncurated (difference = number of excluded samples).
