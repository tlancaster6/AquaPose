    3. Replace `_batch_score_frames()` with `_batch_score_frames_kpt()`:

       ```python
       def _batch_score_frames_kpt(
           batch_frames: list[int],
           frames_a: dict[int, int],
           frames_b: dict[int, int],
           tracklet_a: Tracklet2D,
           tracklet_b: Tracklet2D,
           lut_a: ForwardLUT,
           lut_b: ForwardLUT,
           config: AssociationConfigLike,
       ) -> tuple[float, int]:
       ```

       Returns `(score_sum, n_skipped)` where n_skipped is frames with zero valid keypoints.

       Implementation:
       a. Build idx_a, idx_b from frames_a/frames_b mappings (same as before).
       b. Extract keypoints: `kpts_a = tracklet_a.keypoints[idx_a]` -> (N, K, 2), same for b.
       c. Extract confidences: `conf_a = tracklet_a.keypoint_conf[idx_a]` -> (N, K), same for b.
       d. Build intersection mask: `valid = (conf_a >= config.keypoint_confidence_floor) & (conf_b >= config.keypoint_confidence_floor)` -> (N, K) bool.
       e. Count valid per frame: `n_valid = valid.sum(axis=1)` -> (N,).
       f. Identify active frames: `active_mask = n_valid > 0` -> (N,) bool. Count `n_skipped = int((~active_mask).sum())`.
       g. If no active frames, return (0.0, len(batch_frames)).
       h. Flatten valid keypoints: `pixels_a = kpts_a[valid]` -> (M, 2), same for b. M = valid.sum().
       i. Cast rays: `pix_a_t = torch.tensor(pixels_a, dtype=torch.float32)`, call `lut_a.cast_ray(pix_a_t)`, convert to numpy float64 with `.cpu().numpy()`. Same for camera B.
       j. Compute distances: `dists = ray_ray_closest_point_batch(oa, da, ob, db)` -> (M,).
       k. Scatter back to per-frame means using `np.add.reduceat`:
          - `active_n_valid = n_valid[active_mask]` -> (A,) where A = active frames count
          - `offsets = np.concatenate([[0], np.cumsum(active_n_valid[:-1])])`
          - `sums = np.add.reduceat(dists, offsets)` -> (A,)
          - `mean_dists = sums / active_n_valid` -> (A,)
       l. Apply soft kernel: `contributions = np.where(mean_dists < config.ray_distance_threshold, 1.0 - mean_dists / config.ray_distance_threshold, 0.0)` -> (A,).
       m. Return `(float(contributions.sum()), n_skipped)`.

    4. Update `score_tracklet_pair()`:

          if tracklet_a.keypoints is None or tracklet_b.keypoints is None:
