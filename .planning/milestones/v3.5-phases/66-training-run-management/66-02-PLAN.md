  <files>src/aquapose/training/compare.py, tests/unit/training/test_compare.py, src/aquapose/training/__init__.py</files>
  <behavior>
    - discover_runs: Given a training type directory (e.g. .../training/obb/), returns sorted list of run_* directory Paths
    - discover_runs: Returns empty list if directory does not exist
    - load_run_summaries: Given list of run dirs, loads summary.json from each, returns list of dicts; skips runs with missing summary.json (logs warning)
    - format_comparison_table: Given list of summary dicts, returns string with columns: Run, Tag, mAP50, mAP50-95, Prec, Recall, Epoch, Duration, Sources; best metric values are wrapped in click.style(bold=True, fg="green")
    - format_comparison_table: Returns "No runs found" message for empty input
    - write_comparison_csv: Given list of summary dicts and output path, writes CSV with same columns (no ANSI codes)
  </behavior>
  <action>
Create `src/aquapose/training/compare.py` with:

1. `discover_runs(training_type_dir: Path) -> list[Path]` — List and sort `run_*` subdirectories by name. Return empty list if dir doesn't exist.

2. `load_run_summaries(run_dirs: list[Path]) -> list[dict]` — For each dir, load `summary.json` via json.load. Skip dirs where summary.json is missing (log a warning). Return list of parsed dicts.

3. `format_comparison_table(summaries: list[dict]) -> str` — Build a terminal table string with columns:
   - Run (run_id, truncated)
   - Tag (tag or "-")
   - mAP50 (from metrics.mAP50, format .4f)
   - mAP50-95 (from metrics["mAP50-95"], format .4f)
   - Prec (metrics.precision, .4f)
   - Recall (metrics.recall, .4f)
   - Epoch (metrics.best_epoch)
   - Duration (training_duration_seconds formatted as Xm Ys)
   - Sources (e.g. "80% cons / 20% gap" from dataset_sources, or "-" if no provenance)

   For metric columns (mAP50, mAP50-95, Prec, Recall), find the best value and wrap that cell with `click.style(cell, fg="green", bold=True)`.

   Header row uses `click.style(header, bold=True)`.

   Return "No runs found." if summaries list is empty.

4. `write_comparison_csv(summaries: list[dict], output_path: Path) -> None` — Write same columns as terminal table to CSV (no ANSI codes). Use csv.DictWriter.

```python
    "--config",

)

    "--model-type",
    required=True,
    type=click.Choice(["obb", "seg", "pose"]),
    help="Model type to compare runs for.",
)

    "--csv",
    "csv_path",
    default=None,
    type=click.Path(),
    help="Export comparison to CSV file.",
)
@click.argument("run_paths", nargs=-1, type=click.Path(exists=True))
def compare(config: str, model_type: str, csv_path: str | None, run_paths: tuple[str, ...]) -> None:
    """Compare training runs side-by-side."""
```
