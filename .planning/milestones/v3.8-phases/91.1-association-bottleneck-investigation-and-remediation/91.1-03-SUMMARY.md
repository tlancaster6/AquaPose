# Phase 91.1 Plan 03: Vectorize Recovery Summary

**Vectorized recovery ray-casting (batch cast_ray per camera, batched singleton pixels) and restructured _attempt_split_assign from O(splits*groups*frames) re-scoring to O(groups*frames) precompute + O(splits*groups) numpy sweep**

## Accomplishments
- Batched cast_ray per camera in _triangulate_group_keypoints_for_frame and _triangulate_group_centroid_for_frame
- Batched all singleton pixels into one cast_ray call in _score_singleton_against_group
- Added _point_to_ray_distance_batch for vectorized point-to-ray computation
- Added _compute_per_frame_residuals returning per-frame residual array
- Restructured _attempt_split_assign to precompute (n_groups, n_frames) residual matrix
- Sweep uses numpy nanmean slicing instead of calling _score_singleton_against_group per split
- Precomputed per-group detected frame sets eliminate _slice_tracklet and _has_camera_overlap_with_list in inner loop
