  <behavior>
    Frame selection (select_frames):
    - select_frames((10, 20, 30, ..., 300), n_frames=15) returns 15 deterministic frame indices via np.linspace
    - select_frames with n_frames > available returns all frames and emits a warning
    - select_frames with empty tuple returns empty list
    - Same inputs always produce same outputs (deterministic)

    Tier 1 (compute_tier1):
    - Given a list of (frame_idx, dict[int, Midline3D]) results, produces per-camera and per-fish mean/max reprojection error
    - Handles Midline3D.per_camera_residuals = None gracefully (skips that fish)
    - Overall aggregates computed across all fish and cameras

    Tier 2 (compute_tier2):
    - Given baseline and dropout results, computes max control-point displacement per (fish, dropout_camera)
    - Records None for failed dropout reconstructions
    - Displacement is Euclidean 3D distance: max over 7 control points
  </behavior>
  <action>
    1. Create `src/aquapose/evaluation/__init__.py` with docstring and `__all__` (will be populated as modules are built).

    2. Create `src/aquapose/evaluation/metrics.py` with:

       **select_frames(frame_indices: tuple[int, ...], n_frames: int = 15) -> list[int]:**
       - If empty, return [].
       - If len(available) <= n_frames: return all, warn if strictly fewer than requested.
       - Otherwise: `positions = np.linspace(0, len(available) - 1, n_frames, dtype=int)`, return `[available[int(p)] for p in positions]`.

       **Tier1Result and Tier2Result dataclasses** (frozen, used as return types):
       ```python
       @dataclass(frozen=True)
       class Tier1Result:
           per_camera: dict[str, dict[str, float]]   # cam_id -> {"mean_px": ..., "max_px": ...}
           per_fish: dict[int, dict[str, float]]      # fish_id -> {"mean_px": ..., "max_px": ...}
           overall_mean_px: float
           overall_max_px: float

       @dataclass(frozen=True)
       class Tier2Result:
           per_fish_dropout: dict[int, dict[str, float | None]]  # fish_id -> cam_id -> max displacement (m) or None
       ```

       **compute_tier1(frame_results: list[tuple[int, dict[int, Midline3D]]]) -> Tier1Result:**
       - Accumulate per_camera_residuals from each Midline3D across all frames.
       - Guard: `for cam_id, err in (midline3d.per_camera_residuals or {}).items()`.
       - Per-camera aggregate: mean and max of all errors for that camera across all fish/frames.
       - Per-fish aggregate: mean and max of midline3d.mean_residual across all frames for that fish.
       - Overall: mean and max across everything.

       **compute_tier2(tier2_data: dict[int, dict[str, list[float | None]]]) -> Tier2Result:**
       - Input is pre-accumulated by the harness: fish_id -> dropout_cam -> list of per-frame displacements (or None).
       - For each (fish, cam): if all entries are None, result is None. Otherwise, max of non-None values.


       **_build_models_from_calib(calib_bundle: CalibBundle) -> dict[str, RefractiveProjectionModel]:**
       - For each camera in calib_bundle.camera_ids, construct RefractiveProjectionModel with `K=torch.from_numpy(...)`, etc.
       - Uses `.float()` to ensure float32 tensors.
       - Shared params: water_z, normal=torch.from_numpy(calib_bundle.interface_normal), n_air, n_water.

       **run_evaluation(fixture_path: Path | str, n_frames: int = 15, output_dir: Path | None = None) -> EvalResults:**
       1. Load fixture via `load_midline_fixture(fixture_path)`.
       2. Validate: if `fixture.calib_bundle is None`, raise ValueError with clear message about v1.0 fixtures.
       3. Build models via `_build_models_from_calib(fixture.calib_bundle)`.
       4. Select frames via `select_frames(fixture.frame_indices, n_frames)`.
       5. Build `frame_to_pos = {fi: pos for pos, fi in enumerate(fixture.frame_indices)}` lookup.
       6. **Tier 1 collection:** For each selected frame, get midline_set from `fixture.frames[frame_to_pos[fi]]`, call `triangulate_midlines(midline_set, models, frame_index=fi)`. Collect `(fi, result_dict)` pairs.
       7. **Tier 2 collection:** For each selected frame and each fish, get baseline from step 6 results. For each observing camera (keys in `midline_set[fish_id]`), build reduced midline_set (new dict excluding dropout camera for ALL fish, not just current fish — per the research pseudocode pattern), call `triangulate_midlines(reduced_midline_set, models, frame_index=fi)`. If dropout result has the fish, compute `max_control_point_displacement` via `np.linalg.norm(dropout.control_points - baseline.control_points, axis=1).max()`. Accumulate into `tier2_data[fish_id][dropout_cam].append(displacement_or_None)`.
       8. Compute `tier1 = compute_tier1(frame_results)`, `tier2 = compute_tier2(tier2_data)`.
       9. Format summary via `format_summary_table(...)`.
       10. Determine output_dir: if None, use `fixture_path.parent`. Write JSON via `write_regression_json(... output_dir / "eval_results.json")`.
       11. Return `EvalResults(...)`.
