    - format_comparison_table produces a multi-line string with columns: Stage, Metric, Run A, Run B, Delta, %Change
    - Primary metrics get click.style highlighting: green+bold if improved, red+bold if regressed
    - write_comparison_json writes structured JSON to the later run's directory
