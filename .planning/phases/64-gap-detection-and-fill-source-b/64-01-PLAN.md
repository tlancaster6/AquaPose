Add four functions to `src/aquapose/training/pseudo_labels.py`:

1. **`detect_gaps()`** -- Core gap detection function:
   - Signature: `detect_gaps(midline: Midline3D, inverse_lut: InverseLUT, proj_models: dict[str, RefractiveProjectionModel], frame_detections: dict[str, list], frame_tracks: dict[str, list], min_cameras: int = 3) -> list[tuple[str, str]]`
   - Return early with `[]` if `midline.per_camera_residuals` is None or `len(contributing) < min_cameras`
   - Compute centroid as `midline.control_points.mean(axis=0)` (shape (3,))
   - Call `ghost_point_lookup(inverse_lut, torch.from_numpy(centroid_3d[None].astype(np.float32)))` to get visible cameras
   - Gap cameras = visible_cam_ids - contributing_cam_ids
   - For each gap camera, call `_classify_gap()` and collect `(cam_id, reason)` tuples
   - Import `ghost_point_lookup` from `aquapose.calibration.luts`

2. **`_classify_gap()`** -- Private classification function:
   - Signature: `_classify_gap(cam_id: str, midline: Midline3D, proj_model: RefractiveProjectionModel, frame_detections: dict[str, list], frame_tracks: dict[str, list]) -> str`
   - Project centroid via `proj_model.project(torch.from_numpy(centroid_3d[None].astype(np.float32)))` -- use projected centroid per user decision, NOT LUT pixel coords
   - If projection is invalid, return `"no-detection"`
   - Extract cx, cy from projection result using `.cpu().detach().numpy()` (CUDA safety)
   - Check frame_detections.get(cam_id, []) for any Detection whose bbox (x, y, w, h) contains (cx, cy): `x <= cx <= x + w and y <= cy <= y + h`
   - If no matching detection: return `"no-detection"`
   - Check frame_tracks.get(cam_id, []) for any Tracklet2D that: (a) has `midline.frame_index` in its `frames` tuple, and (b) has `frame_status[idx] == "detected"` at that frame index. Use `frames.index(midline.frame_index)` inside a try/except ValueError since `.frames` is a tuple.
   - If no matching tracklet: return `"no-tracklet"`
   - Otherwise: return `"failed-midline"`

3. **`generate_gap_fish_labels()`** -- Gap-specific label generation:
   - Signature: `generate_gap_fish_labels(midline: Midline3D, projection_model: RefractiveProjectionModel, img_w: int, img_h: int, keypoint_t_values: list[float], lateral_pad: float) -> dict | None`
   - Same logic as existing `generate_fish_labels()` but WITHOUT the per_camera_residuals membership check and WITHOUT the per-camera residual threshold check
   - Call `reproject_spline_keypoints()` to get keypoints_2d and visibility
   - Return None if `visibility.sum() < 2`
   - Call `_passes_bounds_check(keypoints_2d, visibility, img_w, img_h)` -- return None if fails
   - Compute OBB, format OBB line, format pose line (same as generate_fish_labels)
   - Compute confidence via `compute_confidence_score(midline.mean_residual, midline.n_cameras, midline.per_camera_residuals)` -- same fish-level score, no discount per user decision
   - Return dict with keys: `obb_line`, `pose_line`, `confidence`, `raw_metrics`, `keypoints_2d`, `visibility`

4. **`_passes_bounds_check()`** -- Bounds validation:
   - Signature: `_passes_bounds_check(keypoints_2d: np.ndarray, visibility: np.ndarray, img_w: int, img_h: int) -> bool`
   - Extract visible points: `vis_pts = keypoints_2d[visibility]`
   - Return False if `len(vis_pts) < 2`
   - Check `0 <= x < img_w` and `0 <= y < img_h` for each visible point
   - Return True if at least 50% of visible points are in bounds
