The SegmentationBackend class must:

1. **`__init__`**: Accept explicit kwargs: `weights_path: str | None`, `confidence_threshold: float = 0.5`, `n_points: int = 15`, `min_area: int = 300`, `device: str = "cuda"`, `crop_size: tuple[int, int] = (128, 64)`. Store all as instance attributes. If `weights_path` is not None and file exists, load YOLO model: `from ultralytics import YOLO; self._model = YOLO(str(weights_path))`. If weights_path is None, set `self._model = None` and log a warning (allows graceful no-model fallback returning midline=None).

2. **`process_frame`**: For each camera in `camera_ids`, for each detection in `frame_dets[cam_id]`:
   a. Extract affine crop using `extract_affine_crop` from `aquapose.segmentation.crop`. Compute OBB center from `det.bbox` (cx = x + w/2, cy = y + h/2). Compute OBB w/h from `det.obb_points` side lengths if available; fall back to `det.bbox[2], det.bbox[3]` if `obb_points` is None. Use `angle = det.angle if det.angle is not None else 0.0`. Pass `crop_size=self._crop_size`, `fit_obb=True`, `mask_background=True`.
   b. If `self._model is None`, return `AnnotatedDetection(midline=None)`.
   c. Run YOLO inference: `results = self._model.predict(crop.image, conf=self._conf, verbose=False)`.
   d. Extract mask: if `results` and `results[0].masks is not None` and `len(results[0].masks.data) > 0`, get `mask_tensor = results[0].masks.data[0]`. Convert: `mask_np = (mask_tensor.cpu().numpy() > 0.5).astype(np.uint8) * 255`. Resize to crop dimensions with `cv2.INTER_NEAREST` if shape mismatch.
   e. Check mask area: `if np.count_nonzero(mask_np) < self._min_area`, skip (midline=None).
   f. Skeletonize using existing helpers from `aquapose.reconstruction.midline`: `_adaptive_smooth`, `_skeleton_and_widths`, `_longest_path_bfs`, `_resample_arc_length`. Import these directly (do NOT copy the code).
   g. Back-project from crop space to full-frame: `xy_frame = invert_affine_points(xy_crop, crop.M)`. Scale half-widths: `scale = np.sqrt(crop.M[0,0]**2 + crop.M[0,1]**2); hw_frame = (hw_crop / max(scale, 1e-6)).astype(np.float32)`.
   h. Construct `Midline2D(points=xy_frame, half_widths=hw_frame, fish_id=0, camera_id=cam_id, frame_index=frame_idx, is_head_to_tail=False, point_confidence=None)`.
   i. Wrap in `AnnotatedDetection(detection=det, mask=mask_np, crop_region=None, midline=midline, camera_id=cam_id, frame_index=frame_idx)`.

3. Handle all failure cases gracefully: no model, no detections, no masks, mask too small, skeleton too short, empty BFS path -- all return `AnnotatedDetection(midline=None)`.

**CRITICAL coordinate concern:** The YOLO-seg mask from `results[0].masks.data` may not match crop pixel dimensions. Always assert and resize if needed. Use `cv2.INTER_NEAREST` for binary masks (never bilinear).

**CRITICAL import boundary:** Do NOT import from `aquapose.engine` at runtime. Only import from `aquapose.reconstruction.midline`, `aquapose.segmentation.crop`, `aquapose.segmentation.detector`, `aquapose.core.midline.types`.

The PoseEstimationBackend class must:

1. **`__init__`**: Accept explicit kwargs: `weights_path: str | None`, `device: str = "cuda"`, `n_points: int = 15`, `n_keypoints: int = 6`, `keypoint_t_values: list[float] | None = None`, `confidence_floor: float = 0.3`, `min_observed_keypoints: int = 3`, `crop_size: tuple[int, int] = (128, 64)`, `conf: float = 0.5`. Store all. If `keypoint_t_values` is None, default to `np.linspace(0, 1, n_keypoints)`. If `weights_path` is not None and file exists, load YOLO model: `from ultralytics import YOLO; self._model = YOLO(str(weights_path))`. If None, `self._model = None` + warning.

2. **`_keypoints_to_midline` (private method or module-level function)**:
   Takes `kpts_xy: np.ndarray (K, 2)`, `t_values: np.ndarray (K,)`, `confidences: np.ndarray (K,)`, `n_points: int`. Uses `scipy.interpolate.interp1d` with `kind="linear"` and `fill_value="extrapolate"` to interpolate x and y independently over `np.linspace(0, 1, n_points)`. Also interpolates confidence values with `bounds_error=False, fill_value=(conf[0], conf[-1])`. Returns `(xy: np.ndarray (n_points, 2), conf: np.ndarray (n_points,))` as float32.

3. **`process_frame`**: For each camera, for each detection:
   a. Extract affine crop (same pattern as SegmentationBackend -- compute OBB center, dims, angle from detection; use `extract_affine_crop` with `fit_obb=True, mask_background=True`).
   b. If `self._model is None`, return `AnnotatedDetection(midline=None)`.
   c. Run YOLO inference: `results = self._model.predict(crop.image, conf=self._conf, verbose=False)`.
   d. Extract keypoints: if `results` and `results[0].keypoints is not None` and `len(results[0].keypoints.xy) > 0`, get `kpts_xy = results[0].keypoints.xy[0].cpu().numpy()` (shape (6, 2)) and `kpts_conf = results[0].keypoints.conf[0].cpu().numpy()` (shape (6,)).
   e. Filter by confidence: `visible_mask = kpts_conf >= self._confidence_floor`. Count visible: `n_visible = visible_mask.sum()`.
   f. If `n_visible < self._min_observed_keypoints`, skip (midline=None).
   g. Get visible keypoints and their t-values: `visible_kpts = kpts_xy[visible_mask]`, `visible_t = self._keypoint_t_values[visible_mask]`, `visible_conf = kpts_conf[visible_mask]`.
   h. Call `_keypoints_to_midline(visible_kpts, visible_t, visible_conf, self._n_points)` to get `(xy_crop, conf_resampled)`.
   i. Back-project: `xy_frame = invert_affine_points(xy_crop, crop.M)`.
   j. Construct `Midline2D(points=xy_frame, half_widths=np.zeros(self._n_points, dtype=np.float32), fish_id=0, camera_id=cam_id, frame_index=frame_idx, is_head_to_tail=False, point_confidence=conf_resampled)`. Note: pose backend has no distance transform, so `half_widths` are zeros.
   k. Wrap in `AnnotatedDetection`.

4. Handle all failure cases: no model, no detections, no keypoints, too few visible keypoints -- all return `AnnotatedDetection(midline=None)`.

**CRITICAL:** Use `.cpu().numpy()` for all tensor-to-numpy conversions (CUDA tensors).
**CRITICAL:** Use `results[0].keypoints.xy` (pixel coords), NOT `.xyn` (normalized).
**CRITICAL import boundary:** Do NOT import from `aquapose.engine` at runtime.
