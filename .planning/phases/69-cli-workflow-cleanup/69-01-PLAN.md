  <behavior>
    - resolve_project("YH") returns Path("~/aquapose/projects/YH").expanduser() when dir exists
    - resolve_project("nonexistent") raises click.ClickException
    - resolve_project(None) from CWD inside project dir returns that project dir
    - resolve_project(None) from CWD outside any project raises click.ClickException
    - resolve_run(None, project_dir) returns most recent run_* dir (sorted by name)
    - resolve_run("latest", project_dir) returns most recent run_* dir
    - resolve_run("20260306_075925", project_dir) returns matching run dir
    - resolve_run("20260306", project_dir) returns matching run dir (partial timestamp)
    - resolve_run("-1", project_dir) returns latest, "-2" returns second-latest
    - resolve_run("-99", project_dir) with fewer runs raises click.ClickException
    - resolve_run("/abs/path/to/run", project_dir) returns that path if it exists
    - get_project_dir(ctx) lazily resolves and caches in ctx.obj
    - get_config_path(ctx) returns project_dir / "config.yaml"
    - AQUAPOSE_HOME = Path("~/aquapose/projects").expanduser()
  </behavior>
  <action>
    Create `src/aquapose/cli_utils.py` with:

    1. `AQUAPOSE_HOME = Path("~/aquapose/projects").expanduser()` constant.

    2. `resolve_project(name: str | None) -> Path`:
       - If name provided: check `AQUAPOSE_HOME / name` exists, return it or raise click.ClickException.
       - If None: walk `Path.cwd()` upward looking for `config.yaml`, stop at home dir. Raise if not found.

    3. `resolve_run(run_ref: str | None, project_dir: Path, subdir: str = "runs") -> Path`:
       - None/"latest" -> latest run (sorted `run_*` dirs by name).
       - Starts with `-` and rest is digits -> negative index into sorted runs list.
       - Matches `^\d{8}` -> partial timestamp match in runs_dir.
       - Otherwise -> treat as full path, resolve if exists.
       - All error cases raise click.ClickException with descriptive messages.

    4. `get_project_dir(ctx: click.Context) -> Path`: Lazy resolver from ctx.obj["project_name"].
    5. `get_config_path(ctx: click.Context) -> Path`: Returns `get_project_dir(ctx) / "config.yaml"`.

    Internal helpers: `_sorted_runs(runs_dir) -> list[Path]`, `_latest_run(runs_dir) -> Path`.

    Write tests in `tests/unit/test_cli_utils.py` using `tmp_path` to create mock project and run directory structures. Use `monkeypatch` to override CWD for CWD-detection tests.
  </action>
  <verify>
    <automated>hatch run test tests/unit/test_cli_utils.py -x</automated>
  </verify>
